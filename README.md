# UK-Housing-Intelligence-Dashboard
Visual measure of economic and monetary stress in the form of an interactive Power BI dashboard analyzing mortgage dependency and financial vulnerability across the United Kingdom. This project combines long-term housing price trends with mortgage-financed housing activity to identify regions potentially exposed to affordability deterioration and interest-rate sensitivity. 

The dashboard integrates multi-source housing datasets and DAX-based calculated measures to explore regional divergence in UK housing markets. Key analytical areas include housing price growth, mortgage vs cash sales structure, mortgage dependency ratios, and regional affordability risk analysis. 

Key Features
- Interactive Power BI dashboard with dynamic filtering
- Regional housing price trend analysis
- Mortgage vs. cash housing sales comparison
- DAX-based Mortgage Dependency Ratio modelling
- Financial vulnerability and affordability analysis
- Executive insight and economic interpretation layer

Tools and Technologies
- Power BI
- DAX
- CSV/Excel datasets
- Data modelling and visualizations
- Economic and Financial analysis

Key Insights
- London and the South-East exhibit persistently elevated housing prices and affordability pressures.
- Mortgage-financed housing activity dominates across most UK regions, indicating continued household sensitivity to interest-rate changes. 
- Regions with higher mortgage dependency may face greater vulnerability during periods of monetary tightening.
- Long-term housing price divergence has widened significantly across UK regions over time. 

This project was developed as part of a broader interest in economic intelligence, financial vulnerability modelling, and business analytics using Power BI.
